# Installation

## Peer dependencies

React skeleton requires the following peer dependencies to work. Make sure to
have them installed in your project.

<Hint type="info">
  You can **skip this step** if those packages are already installed in your
  application.
</Hint>

```bash
$ npm install react@16.8.x react-dom@16.8.x @emotion/styled@10.x
```

> Submit a <a href="https://github.com/nejcm/react-skeleton/pulls">Pull
> request</a> to bring React skeleton to your framework (e.g.
> styled-components).

## Install

Install the latest version of React skeleton by running the following command in
your project's root directory:

```bash
$ npm install @nejcm/react-skeleton --save
```

## Use

```jsx
import React from 'react';
import Skeleton, {Card, Comment} from '@nejcm/react-skeleton';

const YourComponent = () => {
  return (
    <Skeleton>
      <Skeleton.Rectangle height={20} />
      <Skeleton.Circle width={40} height={40} />
    </Skeleton>
  );
};

export default YourComponent;
```
